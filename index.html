Minor Project


Pupil Heart:Heart Rate Variability(HRV) Monitoring via Pupillary Fluctuations.

Pupil Heart as a computer vision- based mobile HRV monitoring system, including a mobile terminal and a server side. On the mobile terminal, during face recognition, Pupil Heart has collected pupil size information through the front facing camera on mobile phones. On the server side, after preprocessing the raw pupil size data, Pupil Heart has extracted high-dimension features using 1-D convolutional neural network(1DCNN), and based on this, has built a pupil-HRV model by recurrent neural network (RNN) . On that basis, Pupil Heart has achieved daily HRV monitoring. The overall results have shown that Pupil Heart can accurately predict a user’s HRV when unlocking phones using face recognition.

Major Project


Oral Cancer Detection Using Deep Learning.


Oral cancer is a significant public health concern worldwide, often leading to high mortality rates due to late-stage detection. This project presents a deep learning-based solution for detecting oral cancer using advanced convolutional neural network architectures, ResNet152V2 and MobileNet. The models were trained on a carefully curated dataset comprising labeled images of cancerous and non-cancerous oral cavities. This application serves as a user-friendly interface for healthcare professionals and researchers. users can upload images of oral cavities, which are then processed and provide real-time predictions. The results include classifications as "Cancer" or "Non-Cancer".
